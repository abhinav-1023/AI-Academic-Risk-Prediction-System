# AI-Based Academic Risk Prediction System

## 📌 Description

This project predicts whether a student is at academic risk using machine learning techniques.

## ⚙️ Tech Stack

- Python
- Pandas
- Scikit-learn
- NumPy

## 🚀 Features

- Predict student performance
- Identify at-risk students
- Data preprocessing and visualization

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py
